Git

Git is a free and open-source version control system, originally created by Linus Torvalds in 2005.
Unlike older centralized version control systems such as SVN and CVS, Git is distributed: every developer has the full history of their code repository locally.

What is Git?
Git is a DevOps tool used for source code management. It is a free and open-source version control system used to handle small to very large projects efficiently.
